Bauer

Bauer is a tool that automates Business-As-Usual (BAU) tasks using Copilot.

Overview

Bauer parses Google document suggestions, creates a LLM-readable file with instructions and actionable suggestions, then generates a PR automatically through Copilot CLI.

Tech stack

Go, Copilot CLI, GitHub CLI, Google API
