Meowatch is a cat monitoring system built with ***Raspberry Pi 4*** and ***Camera Module 3***.
It is an application that serves live stream video content that includes cat-specific features such as identifying individual cats and detecting its behaviours.

It is accessible through an online dashboard and also provides a summary of the cats' activities.

Tech stack: `Go, Hugo`
